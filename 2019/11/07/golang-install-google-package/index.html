安装golang.org/x下面的包

Nov 07 2019

很多第三方库都用到了golang.org/x下面的包，但是国内因为大家都懂的原因是无法访问到的，
不过没关系，在github.com/golang下面，基本都有对应的包 以安装sys包为例

git clone http://github.com/golang/sys.git $GOPATH/src/golang.org/x/sys

如果我们需要sys包里面的unix包，执行上述操作后，继续操作 go install golang.org/x/sys/unix 这样我们就可以愉快的使用了…….
